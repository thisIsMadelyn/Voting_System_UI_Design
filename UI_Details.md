## UI DESIGN DETAILS
- **Layout:** fixed sidebar with logo, nav groups, active states and a user row at the bottom.
- **KPI cards:** 4 stat cards (Revenue, Active Users, Uptime) with colour-coded accent bars, delta badges and icons
- **Revenue chart:** SVG line chart with gradient fills, dashed expense line, hover tooltips
and two mini sparklines below (Avg Order Value + Conversation Rate).
- **Live activity feed:** event stream with colour-coded dots per event type.
- **Top Users table:** avatar initials status pills (active/pending/paused), LTV column.
- **Traffic Sources:** SVG donut chart with animates segment + breakdown bars.
